## NFL Betting Market Analysis
A historical analysis of NFL betting market behavior using game data from 1980 onward.  
This project evaluates long-term performance of moneyline and spread strategies and identifies potential market inefficiencies. Originally completed in Python; the source code was lost during archival, but the full presentation remains available.
